# SnakePixel 🐍

一个贪吃蛇游戏，包含多种游戏模式、道具系统和丰富的视觉特效。

## 游戏模式

| 模式 | 说明 |
|------|------|
| **经典** | 标准玩法，吃食物得分，撞到自己身体游戏结束 |
| **限时** | 60 秒内尽可能获得高分 |
| **生存** | 初始速度更快，挑战极限 |

## 道具系统

| 道具 | 效果 |
|------|------|
| ● 红色食物 | 普通，+10 分 |
| ★ 金色食物 | +30 分 |
| ⚡ 加速 | 蛇移动速度提升，持续 5 秒 |
| ❄ 减速 | 蛇移动速度降低，持续 5 秒 |
| ×2 双倍分 | 食物得分翻倍，持续 8 秒 |
| 👻 幽灵 | 穿越自身不死亡，持续 6 秒 |
| 🛡 护盾 | 抵挡一次碰撞伤害，持续 4 秒 |

## 操作

| 按键 | 功能 |
|------|------|
| 方向键 / WASD | 控制蛇移动 |
| 空格 | 暂停/继续 |
| 快速连按方向键（3 次/0.6 秒） | 触发短暂加速（1 秒） |

## 特性

- 粒子特效系统（吃食物、游戏结束）
- 60fps 流畅渲染
- 音效反馈
- 速度调节滑块
- 最高分本地持久化
- 响应式布局
